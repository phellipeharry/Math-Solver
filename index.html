Configurações da API

Para habilitar o reconhecimento de escrita manual e resoluções completas com IA, é necessário inserir sua chave da API do Gemini.

Você pode obter uma chave de API **gratuita** acessando o Google AI Studio .

Importante: A chave é armazenada de forma 100% segura apenas no seu próprio navegador (localStorage) e as chamadas são feitas diretamente para os servidores do Google.

Como Usar o Math Solver

1. Desenhar ou Enviar Foto

Na aba **Calculadora & Gráficos**, você pode desenhar equações à mão no rascunho (com suporte a toque/zoom) ou enviar fotos de contas de matemática. A IA irá reconhecer e desenhar o gráfico da função automaticamente!

2. Tutor de Questões Didático

Use a aba **Tutor de Questões** para colar enunciados de exercícios complexos (como estatística, álgebra, problemas lógicos) ou tirar foto direta de uma folha de exercícios. A IA trará a estratégia de raciocínio "Como Pensar" e resolverá a questão de forma clara com gráficos/ LaTeX.

3. Gráficos em Tempo Real

Visualize as funções no **Plano Cartesiano** bidimensional. Dê zoom com a roda do mouse, arraste o plano e configure eixos de forma 100% interativa.

4. API Key

Cadastre sua chave da API do Gemini obtida de graça no AI Studio do Google no botão **API Key** do topo para usufruir de todas as capacidades visuais e explicativas da inteligência artificial.
